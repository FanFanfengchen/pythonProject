## 核心功能模块

### 1. 计算器模块 (`core/my_calculator.py`)
- 提供简单的加法功能
- 函数：`my_adder(x, y)` - 返回两个数的和

### 2. 句子处理模块 (`core/sentence.py`)
- 提供句子的基本操作
- 类：`Sentence`
  - 方法：`letter_count()` - 返回句子的字母数量
  - 方法：`word_count()` - 返回句子的单词数量
  - 方法：`upper()` - 返回句子的大写形式

### 3. 购物清单模块 (`core/shopping_list.py`)
- 提供购物清单的管理功能
- 类：`ShoppingList`
  - 方法：`get_item_count()` - 返回购物清单的商品数量
  - 方法：`get_total_price()` - 返回购物清单的总价格

## 测试

项目包含完整的测试文件，使用Python的`unittest`框架进行测试：

- `tests/test_my_calculator.py` - 测试计算器功能
- `tests/test_sentence.py` - 测试句子处理功能
- `tests/test_shopping_list.py` - 测试购物清单功能

运行测试：
```bash
python -m unittest discover tests
```
