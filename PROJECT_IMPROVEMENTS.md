## 📝 目录说明

| 目录 | 内容 | 学习重点 |
|------|------|----------|
| `Python数据分析/` | 数据分析学习记录 | pandas、numpy、可视化 |
| `学习笔记/` | Python 基础语法笔记 | 变量、数据类型、控制流 |
| `工具库/` | 实用工具函数 | 模块化、代码复用 |
| `旅途的开始/` | 早期练习代码 | Python 基础 |
| `游戏/` | 小游戏项目 | 综合应用、逻辑实现 |
| `图形界面/` | GUI 编程 | tkinter、事件处理 |
| `独立项目/` | 独立项目作品 | 工程化、完整功能 |
| `认证/` | 登录认证练习 | 用户管理、安全 |

## 🔧 依赖安装

```bash
pip install -r requirements.txt
```
